文件夹包含三个语料库处理函数

all_part_speech_pos.py:
	
	总结语料库中出现词语

	求出词语在标注集上的条件概率

label_set.py:

	整理语料库中出现的标注形式

	与北京大学语料库标注集进行融合

Merge.py

	将语料库中词语编排仅一个txt便于处理

get_HMM_arr.py

	用于生成HMM的词性转移概率，词语输出概率

处理的语料库为北大2014语料库，已上传处理后文件，不在上传原文件

提示：运行时需修改本机路径
